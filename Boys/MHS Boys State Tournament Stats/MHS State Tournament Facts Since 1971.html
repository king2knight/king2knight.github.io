MHS State Tournament Facts

 

Since 1971, McPherson boys basketball, regardless of classification, are:

 

 

First in state tournament wins with 67

 

 

First in state tournament appearances with 34

 

 

First in final four state tournament appearances with 30

 

 

First in state championship game finals appearances with 22

 

 

First in state championships with 13
