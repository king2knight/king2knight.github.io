MHS State Tournament Facts

 

Since 1971, McPherson boys’ basketball, regardless of classification, are:

 

 

First in state tournament wins with 75

 

 

First in state tournament appearances with 37

 

 

First in final four state tournament appearances with 33

 

 

First in state championship game finals appearances with 25

 

 

First in state championships with 15
